1 hour worked on. design for catherine: I’m building a 4 wheel robot chassis using an Arduino Nano as the main controller, a DRV8833 motor driver to run four DC motors two per side, and an RPR-0521RS sensor for proximity and light detection using I2C. I’m powering it with an ISDT BattAir 2S LiPo battery, which runs the motors through the DRV8833 and can also power the Nano through VIN or a regulated supply. I’m not using USB for movement only for testing and uploading code so the whole system runs wirelessly off the battery with all grounds connected. I also did a drawing to show where everything is going, pcb and schematic design. I don't know if i should do a LiPo battery or a battery pack...
